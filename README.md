# Conexión entre Python y NEPLAN mediante SOAP Web Services

Este repositorio contiene los códigos y archivos necesarios para establecer una conexión entre Python y NEPLAN utilizando servicios web SOAP.

## Objetivo

El objetivo principal es proporcionar una guía inicial para la configuración y uso de la conexión, empleando las siguientes herramientas:

- Python
- Visual Studio Code
- Zeep (biblioteca SOAP para Python)
- NEPLAN Web Services

## Contenido del repositorio

- Ejemplos de conexión con NEPLAN
- Configuración básica de Web Services
- Ejemplos de consultas y modificación de parámetros
- Scripts en Python para automatización
